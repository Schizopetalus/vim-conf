Exécuter du vimscript

On peut sélectionner les lignes suivantes (en mode visuel) et les exécuter avec la commande :@

:if "on" == "on"
:   echom "toto"
:endif

sinon on peut exécuter tout le buffer avec la commande:

:%y|@"<CR>
(voir sur : https://vi.stackexchange.com/questions/3186/execute-content-of-buffer-as-vimscript )

(note : @" variable qui représente le registre ", idem @r, @', etc...
c'est puissant on peut l'utiliser dans n'importe quelle commande,
par exemple :
:echom @" pour l'afficher )
